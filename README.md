SkillFlow

SkillFlow is a web application designed to help vulnerable youth in Uganda identify relevant skills and access learning opportunities that improve their employability. The platform provides personalized skill recommendations, learning resources, and progress tracking through an intuitive user interface.

 Features

- User registration and authentication
- User profile management
- Personalized skill recommendations
- Skills catalogue
- Learning resources
- Progress tracking
- Responsive web interface

 Tech Stack

Frontend
- React
- HTML
- CSS
- JavaScript

Backend
- Node.js
- Express.js

Database
- Supabase


 Installation

 Clone the repository

```bash
git clone <repository-url>
```

 Install dependencies

```bash
npm install
```

Start the application

```bash
npm run dev
```
